This project contains the theme assignment from theme 9.

In this folder you'll find markdown files and pdf files. The markdown can be knitted to produce the pdf file, but for convenience this already has been done.
There are two files, the logbook and the results. 
	The logbook is an entire summary of the project and includes the exploratory data analysis. 
	The results are separated from the logbook is fully readable on its own.
	
There are two folders, the data and resources.
In data you'll find all the data that has been used for the machine learning process. The made .arff file is also included and can be run with Weka.

The resources folder contains images and .xlsx files.
These have been used in order to complete the logbook file and include parts of the machine learning process.
